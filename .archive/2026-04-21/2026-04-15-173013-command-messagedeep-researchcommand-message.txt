  研究结论

  我直接读取了 affaan-m/everything-claude-code 仓库的原始内容，以下是完整答案：

  ---
  deep-research Skill 本身不依赖其他 Skills

  它的唯一依赖是 MCP 服务（非 skills）。

  ---
  必须配置的 MCP 服务（二选一，两者都装效果最佳）

  ┌─────────┬────────────────────────────────────────────┬──────────────────┐
  │  MCP    │                 提供的工具                 │   API Key 来源   │
  │  服务   │                                            │                  │
  ├─────────┼────────────────────────────────────────────┼──────────────────┤
  │ firecra │ firecrawl_search、firecrawl_scrape、firecr │ https://firecraw │
  │ wl      │ awl_crawl                                  │ l.dev            │
  ├─────────┼────────────────────────────────────────────┼──────────────────┤
  │ exa     │ web_search_exa、web_search_advanced_exa、c │ https://exa.ai   │
  │         │ rawling_exa                                │                  │
  └─────────┴────────────────────────────────────────────┴──────────────────┘

  仓库 mcp-configs/mcp-servers.json 中的标准配置：

  // ~/.claude.json 的 mcpServers 段落
  "firecrawl": {
    "command": "npx",
    "args": ["-y", "firecrawl-mcp"],
    "env": { "FIRECRAWL_API_KEY": "YOUR_FIRECRAWL_KEY_HERE" }
  },
  "exa-web-search": {
    "command": "npx",
    "args": ["-y", "exa-mcp-server"],
    "env": { "EXA_API_KEY": "YOUR_EXA_API_KEY_HERE" }
  }

  ---
  可选的配套 Skills（非必须，但推荐一并安装）

  └──────────────────────┴───────────────────────────────────────────────────┘

  总结

  安装 deep-research skill 只需两步：
  1. 安装 skill 本体（skills/deep-research/SKILL.md）
  2. 配置 firecrawl 和/或 exa MCP 服务并填入 API Key

  无需安装任何其他 skill 作为前提依赖。
