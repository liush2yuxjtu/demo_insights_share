为什么做这件事（PM 一句话）

痛点：每位工程师都得花 60+ 秒让 claude /insights 从零悟出队友早就悟出的同一条教训；insight 只在个人 profile 里，不会自动流到下一个踩坑的人手上。
修复：LAN 上跑一个 insight wiki（HTTP daemon，端口 7821），加 热加载 + 后台 adapter + 静默 Stop hook。Bob 即使没主动求助，只要在 claude code 里描述一句问题，系统就会在用户无感知的时间里把 Alice 已经写下的 insight 卡片悄悄送到他眼前。
