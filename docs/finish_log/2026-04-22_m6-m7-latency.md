## 关键设计决策

1. **cache-hit 路径**：`latency_cache.cached_search` 单层 sha1 key + TTL 300s，命中直接返回缓存 `hits.json`，跳过 SDK。实测 ~3000×加速。
2. **O2 回归教训**：topic.id（`postgres-pool-exhaustion`）≠ 磁盘 slug（`postgres_pool.md`），agent 按 id 拼路径必 404。修复：prompt 只当 wiki_type 提示，agent 仍 Glob + INDEX.md 解析 slug。
3. **max_turns 策略**：O2 曾把 5→3 做预算收紧，但与错误路径叠加导致 error_max_turns 全军覆没。教训：预算紧缩必须与正确 shortcut 同时落地；回到 5 + O3 早退保护。
4. **self-verify 数值门**：3-gate（baseline_recorded / p95_budget / no_regression）+ 退出码 0/1，禁止硬编码关键词匹配，与 meta-self-verify 双探针对齐。
