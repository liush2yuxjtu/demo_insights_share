# CLAUDE.md 规则格式

## 规则

向本项目 `CLAUDE.md` 添加任何规则时，必须且只能使用以下单行表格格式：

```
| 规则名 | 描述 | 触发时机 | [文件名](docs/rules/文件名.md) |
```

## 要求

- **CLAUDE.md 只保存一行**：表格中每条规则仅占一行，描述保持简洁。
- **全文写入 docs/rules/*.md**：规则的完整说明、示例、反例均写在对应的 md 文件中。
- **禁止以下写法**：散文段落、项目符号列表、独立章节标题、内联展开的详细说明。

## 示例

### 正确

```markdown
| 仅使用中文 | 所有文档、对话、代码注释仅使用中文 | 始终 | [language.md](docs/rules/language.md) |
```

### 错误

```markdown
## 语言规则

- 所有文档、对话、代码注释仅使用中文
```

## 原因

- CLAUDE.md 是规则索引，保持简短才能快速加载进上下文。
- 详情分离到 docs/rules/*.md，便于单独更新、版本追踪和按需加载。
